# Operador de negación (!!)
## Convierte cualquier valor a un boolean
## Ejemplo:
- const preview = 'https://audio.mp3';
- console.log(!!preview); // true
- const preview = '';
- console.log(!!preview); // false


# try...catch
## Permite capturar y manejar errores que pueden ocurrir durante la ejecución de un bloque de código, evitando que la aplicación falle de forma inesperada
## Sintaxis:
try {
  // Código que puede generar un error
} catch (error) {
  // Manejo del error
}
