# En GameService, en el método getTrackLyrics() se implementó lo de guardar en el caché
## Antes, cada consulta de la letra requería una nueva petición a la API, lo que ocasionaba tiempos de espera perceptibles. Al almacenar las respuestas en memoria, las consultas repetidas se resuelven casi de forma instantánea, mejorando la experiencia del usuario y reduciendo el número de solicitudes HTTP

# tap()
## Operador de RxJS que permite ejecutar acciones secundarias sobre los datos emitidos por un Observable sin modificar su contenido

# takeUntil() 
## Operador de RxJS que finaliza automáticamente una suscripción cuando otro Observable emite un valor. Se utiliza para evitar fugas de memoria al cancelar suscripciones que ya no son necesarias

# split() 
## Divide una cadena de texto en un arreglo utilizando un separador

# filter() 
## Crea un nuevo arreglo con los elementos que cumplen una condición

# slice() 
## Devuelve una porción de un arreglo sin modificar el original

# join() 
## Une los elementos de un arreglo en una sola cadena de texto utilizando un separador.
